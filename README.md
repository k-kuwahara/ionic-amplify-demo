# ionic-amplify-demo

## Usage

```bash
# global install amplify cli
$ npm i -g @aws-amplify/cli

# move app's root and install modules
$ cd path/to/ionic-amplify-demo
$ npm install 

# or

$ yarn install

# setup amplify
$ amplify init

# add plugins
$ amplify auth add
$ amplify storage add

# push configuration
$ amplify push

# boot app
$ npm run start

# or

$ yarn start
```
